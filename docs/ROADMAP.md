# Roadmap

Future enhancements planned for post-hackathon development.

## High Priority

| Feature | Description |
|---------|-------------|
| OpenTelemetry Integration | Export metrics to Prometheus/Jaeger |
| Order Execution | REST API integration for trading |
| Python Bindings | pybind11 wrappers |
| Latency Percentiles | P50/P99 tracking |

## Medium Priority

| Feature | Description |
|---------|-------------|
| Time-Travel Replay | Record and replay market data |
| Rate Limit Handling | Auto-throttle on limits |
| Latency Histogram | Distribution API |

## Low Priority

| Feature | Description |
|---------|-------------|
| WebAssembly | Browser-based usage |
| Multi-Exchange | Abstract exchange interface |
